3.4 JDK 动态代理 vs CGLIB

特性 JDK 动态代理 CGLIB
要求 实现接口 继承类
原理 反射 字节码生成
性能 JDK8 后更快 生成类有开销
限制 只能代理接口 不能代理 final

Spring 选择策略：

  • 有接口：JDK 动态代理（默认）
  • 无接口：CGLIB
  • 可配置：spring.aop.proxy-target-class=true 强制 CGLIB

四、事务管理

4.1 事务传播行为

传播行为 说明
REQUIRED（默认） 有事务加入，无事务新建
REQUIRES_NEW 总是新建事务，挂起当前事务
SUPPORTS 有事务加入，无事务非事务执行
NOT_SUPPORTED 非事务执行，挂起当前事务
MANDATORY 必须在事务中，否则抛异常
NEVER 必须非事务，否则抛异常
NESTED 嵌套事务（保存点）
