Q3: CopyOnWriteArrayList 适用场景？

A3: 推荐回答

适用：

  • 读多写少（如配置列表、监听器列表）
  • 遍历操作远多于修改操作
  • 容忍内存占用

不适用：

  • 写操作频繁
  • 数据量大（复制开销大）
  • 需要强一致性
